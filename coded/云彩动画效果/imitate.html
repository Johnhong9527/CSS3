说明: 类名为sky的div标签为父元素,其中定义了position的值为relative, 是为了其内三个子元素可以脱离文档流,也是动画执行的前置条件. 而overflow: hidden;的意思的超出固定宽高部分的内容隐藏. 这是为了避免动画执行的过程出现卡顿现象; 最后是z-index: -80;,目的使该动画以背景图的形象出现. 
.sky {
  background-color: #007fd5;
  position: relative;
  overflow: hidden;
  z-index: -80;
}

更具体的属性说明
.sky .clouds_one {
  背景图
  background: url("../images/cloud_one.png") no-repeat;
  这里设置动画的 keyframe 执行时间 动画从头到尾的速度是相同 规定动画应该无限次播放
  -webkit-animation: cloud_one 30s linear infinite;
  -o-animation: cloud_one 30s linear infinite;
  animation: cloud_one 30s linear infinite;

  -webkit-transform: translate3d(0, 0, 0);
  -ms-transform: translate3d(0, 0, 0);
  -o-transform: translate3d(0, 0, 0);
  transform: translate3d(0, 0, 0);
}

@keyframes cloud_one {
  0%{left:0;}
  100%{left: -200%;}
}
